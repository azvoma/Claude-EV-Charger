# EV Charger Installation Near Me
### UK's Commercial EV Charger Installation Directory
**[evchargerinstallationnearme.co.uk](https://evchargerinstallationnearme.co.uk)**

---

## About
A complete UK directory of 1,136 OZEV-approved commercial EV charger installation companies. Built as a lead generation platform connecting businesses with certified EV charging installers.

## Features
- 🔍 **1,136 verified companies** from across the UK
- 📍 **17 location pages** — London, Manchester, Glasgow, Bristol and more
- ⭐ **Enhanced company profiles** — 5 tabs: Overview, Specs, Brands, Coverage, Reviews
- 🏅 **OZEV accreditation** display with registration numbers
- ⚡ **Technical specs** — phase capability, solar integration, V2G, smart charging
- 💷 **Pricing transparency** — standard install costs, WCS grant deduction
- 📋 **Process steps** — DNO notification, survey method, grant management
- 📱 **Mobile responsive** with hamburger menu
- 🔒 **Lead generation forms** — hero, modal, profile, contact
- 📊 **Full SEO** — Schema.org, Yoast-style meta, sitemap, robots.txt

## Tech Stack
- Pure HTML/CSS/JavaScript — no frameworks, no build tools
- Single-file deployment (index.html)
- All 1,136 company records embedded as JSON

## Deploy in 2 Minutes (Netlify)
1. Fork/clone this repo
2. Go to [netlify.com](https://netlify.com) → **Add new site → Import from Git**
3. Select this repo — build command: *(leave blank)*, publish directory: `.`
4. Click **Deploy**
5. Add custom domain: `evchargerinstallationnearme.co.uk`

## Deploy via Drag & Drop
1. Download as ZIP
2. Go to [app.netlify.com](https://app.netlify.com) → drag the folder into the deploy zone
3. Done ✅

## Submit to Google
After deploying, submit `sitemap.xml` at:  
[search.google.com/search-console](https://search.google.com/search-console)
